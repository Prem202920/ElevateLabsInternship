# Task 12: Image Gallery with Modal Popup

## Overview

This task demonstrates creating an interactive image gallery with a modal popup using HTML, CSS, and JavaScript.

## Features

- Responsive grid-based image gallery
- Click image to open modal popup
- Dynamic modal image loading
- Background overlay focus
- Close modal via close button, outside click, or ESC key
- Background scroll disabled while modal is open

## Technologies Used

HTML5, CSS3 (Grid), JavaScript (DOM)
