# Task 11: Responsive Layout Using CSS Grid

## Overview

This task demonstrates creating a fully responsive layout using CSS Grid with header, sidebar, main content, and footer.

## Features

- CSS Grid with named grid areas
- Responsive design for mobile, tablet, and desktop
- Sidebar navigation
- Combination of CSS Grid and Flexbox
- Clean and readable layout

## Technologies Used

HTML5, CSS3 (Grid & Flexbox)
