# Task 14: Login UI Using LocalStorage (Frontend Only)

## Overview

This task demonstrates a basic frontend authentication flow using LocalStorage.

## Features

- User registration and login UI
- Credentials stored in LocalStorage (demo purpose only)
- Login validation against stored data
- Protected dashboard page
- Logout functionality with session clearing
- Redirect if user is not authenticated

## Important Note

This is **frontend-only authentication** and **not secure for real applications**.

## Technologies Used

HTML5, CSS3, JavaScript, LocalStorage
