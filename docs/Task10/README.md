# Task 10: Consume a Public API Using Fetch API

## Overview

This task demonstrates fetching data from a public API using JavaScript Fetch API and displaying it dynamically on a web page.

## API Used

- Random Quotes API (https://api.quotable.io)

## Features

- Fetch data using Fetch API
- Convert response to JSON
- Display data dynamically using DOM manipulation
- Loading indicator while fetching data
- Error handling using catch
- Console logging for debugging

## Technologies Used

HTML5, CSS3, JavaScript (Fetch API)
