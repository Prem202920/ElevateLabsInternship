# Task 9: Client-Side Form Validation Using JavaScript

## Overview

This task demonstrates custom client-side form validation using JavaScript without relying on HTML default validation.

## Features

- Registration form with name, email, password, confirm password
- HTML validation disabled using novalidate
- JavaScript validation using conditions and regex
- Inline error messages under inputs
- Real-time validation using keyup events
- Error and success border styling
- Form submission prevented until all fields are valid

## Technologies Used

HTML5, CSS3, JavaScript
