# Task 8: To-Do List Application (CRUD)

## Overview

This task demonstrates building a functional To-Do List app using Vanilla JavaScript with full CRUD operations.

## Features

- Add new tasks
- Mark tasks as completed
- Added Calender
- Delete tasks
- Prevent empty task submission
- Dynamic DOM manipulation without page reload

## Technologies Used

HTML5, CSS3, JavaScript
